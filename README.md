# Face Guard — Система распознавания лиц с уведомлениями

## Описание проекта

**Face Guard** — это веб-приложение для мониторинга с помощью камер с использованием компьютерного зрения. 

Система в реальном времени:
- Распознаёт лица на нескольких камерах одновременно
- Отмечает известных людей зелёной рамкой, а неизвестных — красной
- При обнаружении неизвестного лица отправляет уведомление с фото и видео в Discord
- Имеет удобный веб-интерфейс для просмотра всех камер

## Основные возможности

- Поддержка нескольких камер одновременно
- Распознавание лиц с помощью библиотеки `face_recognition`
- Уведомления с фото и видео
- Настраиваемые параметры (чувствительность, длительность видео и т.д.)
- Сохранение всех событий в папку `alerts`

## Технологии

- **Backend**: Python, Flask, Flask-SocketIO
- **Компьютерное зрение**: OpenCV, face_recognition
- **Frontend**: HTML, CSS, JavaScript
- **Уведомления**: Discord Webhook

## Как запустить проект

1. **Склонируйте репозиторий**:
   ```bash
   git clone https://github.com/wloudy/face-guard.git
   cd face-guard
2. Создайте виртуальное окружение (рекомендуется):
    ```bash
    python -m venv .venv
    .venv\Scripts\activate

3. Установите зависимости:
   ```bash
   pip install -r requirements.txt

4. Подготовьте известные лица:
- Положите фото известных людей в папку known_faces/
- Название файла будет использоваться как имя человека (например: mama.jpg)

5. Настройте конфигурацию:
- Откройте файл config.py и вставьте свой Discord Webhook URL

6. Запустите приложение:
    ```bash
    python app.py

7. Откройте в браузере: http://127.0.0.1:5000
